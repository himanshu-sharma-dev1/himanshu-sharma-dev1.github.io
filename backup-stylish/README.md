# Stylish Portfolio Backup

This folder contains the backup of the stylish/animated portfolio design.

## Files
- `index.html` - HTML structure with animations
- `style.css` - Dark theme, gradients, glassmorphism
- `script.js` - Typing animation, theme toggle, scroll effects

## To Restore
```bash
cp backup-stylish/index.html index.html
cp backup-stylish/style.css style.css
cp backup-stylish/script.js script.js
git add . && git commit -m "Restore stylish portfolio" && git push
```

## Features of Stylish Version
- Dark/Light theme toggle
- Typing code animation
- Glassmorphism cards
- Gradient backgrounds
- Smooth scroll animations
- Fade-in effects
